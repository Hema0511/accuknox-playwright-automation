# AccuKnox Playwright Automation Task

## Tech Stack
- Playwright
- JavaScript
- Page Object Model (POM)

## Test Scenarios Automated
1. Login to OrangeHRM
2. Navigate to Admin Module
3. Add New User in Admin Module

## Project Structure
- pages/ → Page Object classes
- tests/ → Test cases
- playwrightconfig.cjs → Playwright configuration

## How to Run the Tests

1. Install dependencies:
   npm install

2. Run tests:
   npx playwright test

3. View report:
   npx playwright show-report
